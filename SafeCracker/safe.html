How to Play

  1. Choose a level of difficulty by entering the number of digits in the safe combination.
  2. Click the Start button.
  3. Guess the numbers by clicking on the little arrow-y selector things in each of the spaces beside the numbers. The button should now say "Check the Combination."
  4. Click the "Check the Combination" button.
  5. If the space around the number turns green, that number is correct. If it turns blue, the number is too low. If it turns red, the number is too high. Adjust the numbers and click the "Check the Combination" button again.

The object of the game is to guess the combination in as few tries as possible.
